IMAGES DU PERSONNAGE (Character)
================================

SYSTEME HUD : UNE SEULE IMAGE PLEIN ECRAN par etat du joueur. L'image
contient les mains (et tout ce qui va avec : bras, pouces, doigts)
positionnees au bon endroit dans le bas de l'ecran. Le reste de
l'image est TRANSPARENT.

Le HUD est etire pour remplir toute la zone du widget (== plein ecran).

----------------------------------------------------------------
IMAGES ATTENDUES (une par etat)
----------------------------------------------------------------
    HandHUD.png        -> Etat 'haut' (paumes vers le haut, par defaut)

D'autres etats peuvent etre ajoutes dans HUD_IMAGES de
src/widgets/player_hands.py (ex. main qui tient un outil, etc.).

----------------------------------------------------------------
CONSEILS
----------------------------------------------------------------
- Format : PNG, fond TRANSPARENT.
- Taille de REFERENCE : 1080 x 2340 (resolution telephone portrait),
  ou 2340 x 1080 si tu prefers travailler en "tall x wide". L'image est
  etiree pour remplir le widget : assure-toi qu'elle a le bon aspect
  ratio (~ 9 : 19.5).
- Les mains sont positionnees dans le BAS de l'image (le reste est
  transparent : ciel/decor visibles a travers).
- Apres ajout : pousse le projet (envoyer-sur-github.bat) pour que
  l'image soit incluse dans l'APK.

Logique des mains : src/widgets/player_hands.py
