IMAGES DU PERSONNAGE (Character)
================================

Depose ici les images du joueur en vue 1re personne : les MAINS, les DOIGTS,
les POUCES et les AVANT-BRAS. Le jeu cherche automatiquement un fichier
nomme d'apres la piece :

    assets/Character/<nom>.png      (ou .jpg / .jpeg)

Tant qu'aucune image n'existe pour une piece, le jeu la dessine au canvas
par defaut. Tu peux donc ajouter les images une par une.

----------------------------------------------------------------
PRINCIPE : UNE SEULE MAIN A FOURNIR (la GAUCHE)
----------------------------------------------------------------
Toutes les images sont concues pour la MAIN GAUCHE (avec l'artwork
positionne a GAUCHE du canvas, le reste transparent). Le code FLIPPE
automatiquement l'image en horizontal pour faire la main DROITE.
=> Tu n'as donc qu'a generer 14 images (7 cote main + 7 cote paume),
   pas 28.

----------------------------------------------------------------
NOMS DE FICHIERS ATTENDUS (exactement, en minuscules)
----------------------------------------------------------------

--- COTE MAIN (vue actuelle paume vers le haut) ----------------
    hand.png            -> Corps de la main (paume + pouce + poignet)
    avant_hand.png      -> Avant-bras (supination)
    doigt1.png          -> Phalange PROXIMALE (base) d'un doigt
    doigt2.png          -> Phalange MOYENNE
    doigt3.png          -> Phalange DISTALE (bout, avec pulpe)
    pouce1.png          -> Phalange PROXIMALE du pouce
    pouce2.png          -> Phalange DISTALE du pouce

--- COTE PAUME (vue dos de main, pour usage futur) -------------
    backhand.png        -> Dos de la main
    avant_backhand.png  -> Avant-bras (pronation)
    backdoigt1.png      -> Phalange proximale (dos)
    backdoigt2.png      -> Phalange moyenne (dos)
    backdoigt3.png      -> Phalange distale (dos, avec ongle)
    backpouce1.png      -> Phalange proximale du pouce (dos)
    backpouce2.png      -> Phalange distale du pouce (dos, avec ongle)

Pour l'instant le jeu n'affiche que la vue COTE MAIN ; les images
"back*" sont la pour quand on ajoutera la vue dos de main.

----------------------------------------------------------------
REUTILISATION DES IMAGES (cote main)
----------------------------------------------------------------
- doigt1/2/3 sont les 3 phalanges d'UN doigt et sont REUTILISEES
  pour les 4 doigts de chaque main : 8 doigts dessines, 3 images.
  Seule la longueur de chaque doigt change.
- pouce1/2 sont les 2 phalanges d'UN pouce, reutilisees pour les 2 pouces.
- hand.png contient TOUTE la paume (corps + thenar + hypothenar + lignes) :
  une seule image remplace tout le canvas decompose.

----------------------------------------------------------------
CONSEILS
----------------------------------------------------------------
- Format conseille : PNG carre 256x256 ou 512x512, fond TRANSPARENT.
  (Puissance de 2 = meilleures perfs sur Android.)
- L'artwork doit etre POSITIONNE A GAUCHE du canvas (pas centre) : le
  flip horizontal automatique pour la main droite suppose cette convention.
- Vue depuis le joueur, paumes vers le haut : pouce vers l'EXTERIEUR de
  l'ecran, auriculaire vers l'INTERIEUR.
- Reflets / ombres directement dans l'image (le PBR n'est pas applique
  aux images chargees).
- Apres avoir ajoute des images : pousse le projet (envoyer-sur-github.bat)
  pour qu'elles soient incluses dans l'APK.
