IMAGES DES OBJETS
=================

Depose ici les images des objets du jeu. Le jeu cherche automatiquement
un fichier nomme d'apres l'objet :

    assets/items/<nom>.png      (ou .jpg / .jpeg)

Tant qu'aucune image n'existe pour un objet, le jeu affiche un "?" avec le
nom dessous. Des que tu ajoutes le bon fichier (meme nom), il s'affiche.

----------------------------------------------------------------
NOMS DE FICHIERS ATTENDUS (exactement, en minuscules)
----------------------------------------------------------------
Objets trouves :
    branche.png            (commun)
    pierre.png             (commun)
    feuille.png
    herbe.png
    champignon.png
    baie.png
    fleur.png
    poisson.png
    roseau.png
    minerai.png
    plume.png
    os.png
    coquillage.png
    carcasse.png           (rare)

Remarque : chaque objet a une RARETE (poids dans src/items.py). Les objets
communs (branche, pierre, herbe...) sortent souvent ; les rares (carcasse...)
sortent peu. De plus, chaque case n'offre qu'un stock limite de trouvailles
(entre 5 et 15) : une fois epuisee, explorer n'y donne plus rien.

Objets fabriques (craft) :
    couteau.png
    hache.png
    lance.png
    feu_de_camp.png        (underscore _ a la place des espaces)

----------------------------------------------------------------
CONSEILS
----------------------------------------------------------------
- Format conseille : PNG carre avec fond transparent (ex. 256x256).
- Le nom doit correspondre EXACTEMENT (minuscules, sans accents,
  underscore pour les espaces).
- Apres avoir ajoute des images : pousse le projet (envoyer-sur-github.bat)
  pour qu'elles soient incluses dans l'APK.

La liste des objets et des recettes est dans : src/items.py
